# Blokki Website

Landing page for Blokki app, inspired by brainrot app design.

## Structure

- `index.html` - Main HTML file
- `styles.css` - All styles and responsive design
- `README.md` - This file

## Features

- Modern, dark-themed design
- Responsive layout (mobile, tablet, desktop)
- Hero section with call-to-action buttons
- Features section highlighting main app capabilities
- Smooth animations and hover effects
- Gradient accents matching app branding

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --bg-primary: #0a0a0f;
    --bg-secondary: #141420;
    --accent-primary: #8b5cf6;
    --accent-secondary: #ec4899;
}
```

### Links
Update App Store and Android waitlist links in `index.html`:
- Line 33: App Store link
- Line 36: Android waitlist link
- Line 100: App Store link (CTA section)
- Line 103: Android waitlist link (CTA section)

### Content
Edit text content directly in `index.html`:
- Hero title and subtitle
- Feature cards descriptions
- Footer links

## Deployment

1. Upload `index.html` and `styles.css` to your web hosting
2. Ensure both files are in the same directory
3. Update all links to point to your actual App Store listing and waitlist

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)
- Responsive design for all screen sizes
